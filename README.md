# SocNet

## ქართულად

SocNet არის JavaFX-ზე შექმნილი მცირე სოციალური ქსელი.

### ფუნქციები

- ავტორიზაცია, რეგისტრაცია და პაროლის აღდგენა
- პოსტების ნახვა, შექმნა, გადაზიარება და წაშლა
- მეგობრების დამატება/წაშლა და შეტყობინებების გაგზავნა
- მომხმარებლების დაბლოკვა/განბლოკვა
- პროფილის პარამეტრები
- თამაშების გვერდი

### ტექნოლოგიები

- Java 25 (Maven)
- JavaFX
- MySQL
- CSS (for FXML)

### გაშვება

1. აუცილებელია გქონდეთ დაყენებული JDK (Java Dev Kit) (https://www.oracle.com/java/technologies/downloads)
2. აუცილებელია გქონდეთ MySQL დაყენებული (https://www.mysql.com/downloads/)

### მონაცემთა ბაზა

აპლიკაცია პირველად ჩართვისას ქმნის ავტომატურად საჭირო ბაზას და ცხრილებს.
(საჭიროა მიუთითოთ თქვენი MySQL კონფიგურაცია [port, user, password])

---

## English

SocNet is a small JavaFX social network app

### Features

- Login, registration and password recovery
- View, create, share and delete posts
- Add friends and messages
- Block users
- Profile settings
- Games

### Tech Stack

- Java 25 (Maven)
- JavaFX
- MySQL
- CSS

### Run
1. Install JDK and MySQL

### Database

After run, the application creates the required database and tables automatically.
